# Replit-Paper-Trading-Application
A paper trading application built within Replit. This application uses BS4 to scrape stock prices and Tkinter as the UI for the application. Though it has a UI it requires the console to perform most of the actions.

#### LINK - https://replit.com/@2ant6/Replit-Paper-Trading

# Features
- Add Ticker using Google Finance URL
- Buy/Sell using live scraped prices
- Activity Log to view all past buy/sell actions
- Win/Loss which tells you your current w/l on all positions
- User Interface Using Tkinter
- Feature to change background and foreground color using hex codes or name (if supported by Tkinter)
